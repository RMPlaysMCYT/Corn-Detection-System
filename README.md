# Frameworks used
- Flutter
- Python 3.11.9
- Visual Studio (For Testing on Windows, **REQUIRES** Desktop App Development With C++)
- Tensorflow
- GCC
- XCode if on MacOS/iOS
- For Linux Here's what it requires tho, but also works for IoT such as Raspberry Pi Tho:
    * clang
    * cmake
    * ninja-build
    * pkg-config
    * libgtk-3-dev
    * libstdc++-12-dev
- Android Testing:
    * Android SDK Build-Tools
    * Android SDK Command-line Tools
    * Android Emulator
    * Android SDK Platform-Tools
    * CMake
    * NDK (Side by side)
